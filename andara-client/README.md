# Andara Client

React frontend for Andara's World.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The application will start on `http://localhost:3000`

## Building

```bash
npm run build
```

## Testing

```bash
npm test
```

## Project Structure

- `src/api/` - API client
- `src/store/` - Redux store and slices
- `src/components/` - React components
- `src/game/` - Game engine (WebGL renderer - deferred)
- `src/types/` - TypeScript type definitions
- `src/utils/` - Utility functions
